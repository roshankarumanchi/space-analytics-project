# 🚀 Space Analytics Project

## Temporal and Geospatial Analysis of Rocket Launches, 
## Payload Deployment and Mission Success (1957–2025)

## 📊 Project Overview
A comprehensive data-driven analysis of global space activity 
using Python, MongoDB and PostgreSQL.

## 📁 Datasets
Download datasets from:
- Launch History: https://planet4589.org/space/gcat
- Satellites: https://www.ucsusa.org/resources/satellite-database
- NASA API: https://api.nasa.gov

## 🛠️ Tech Stack
- Python, Jupyter Notebook
- MongoDB, PostgreSQL
- pandas, plotly, matplotlib
- SQLAlchemy, pymongo

## 🚀 How to Run
1. Clone this repository
2. Download datasets from links above
3. Install requirements: pip install -r requirements.txt
4. Run notebooks in order
